Coded text shortcuts

Navigation

Tab / Shift+Tab
Cycle Tracks → Excerpts → Codebook → Back link
↓ (from page load)
Enter Tracks at row 1

Tracks (source list)

↑ / ↓
Move cursor; excerpts column updates live
Shift+← / Shift+→
Same as ↑ / ↓ (works from any zone)
Home / End
First / last track
Space
Pin / unpin the cursor's track
Shift+↑ / Shift+↓
Range-select into the pinned set
Cmd/Ctrl+A
Pin all tracks

Excerpts

↑ / ↓
Move through excerpts; matching source track highlights
Home / End
First / last excerpt

Codebook

↑ / ↓
Move between codes
Home / End
First / last code
Enter
Open that code's coded-text view
/
Focus the codebook search input

Other

?
Open this cheat sheet
N
Back to coding with notes drawer open
Esc
Clear filters / selection; second press navigates back
