ACE v2 — Revised Architecture

Updated after expert audit. Changes from original proposal highlighted.

What Changed After Audit

Key Revisions
Removed schemas/ directory — Pydantic adds nothing for a local HTMX app with form data, no JSON API consumers
Removed fragments/ directory — use Jinja2 macros + block rendering instead. 6 template files, not 20
Merged 6 route files → 4: pages.py, api_coding.py, api_codebook.py, api_project.py
Merged models/coder.py + models/project.py → one file (37 LOC combined)
Deleted services/icr.py — strict subset of agreement_computer.py
Deleted scikit-learn dependency (50 MB) — replace with 15-line Cohen's kappa function
Deleted Direct pandas usage — use stdlib csv + openpyxl directly
Added idiomorph extension (~4 KB) for DOM-diffing text panel swaps (preserves scroll position)
Added UTF-16 offset conversion moves to bridge.js (browser creates UTF-16, convert before POSTing)
Split CSS: ace.css (shared tokens + components) + coding.css (annotation/DnD-specific)

State Management — Three Tiers

Where State Lives
Server session current_index, coder_id, project_path In-memory dict (single user, local app)
Server app.state UndoManager (per source, ephemeral) Lost on restart, by design
JS variables pending_selection (text offsets), reorg_mode, sort toggle Sent to server via hx-vals when needed
localStorage collapsed groups, grid visibility, coder name Survives page reload, replaces app.storage

HTMX Patterns — Coding Page

How cross-pane updates work (OOB swaps)
User clicks code to apply annotation: POST /api/coding/annotate body: {code_id, start, end, text, source_id} Server response (single HTTP response, multiple swap targets): <div id="text-panel"> ...re-rendered annotated text... </div> <div id="annotation-list" hx-swap-oob="true"> ...updated annotation list... </div> HTMX swaps #text-panel (primary target) AND #annotation-list (out-of-band) in one response. No race conditions, no flicker, fully inspectable in Network tab.

Dependency Reduction

Before (NiceGUI)
  • nicegui (huge — Vue, Quasar, WebSocket, 40+ transitive deps)
  • pandas (~30 MB)
  • scikit-learn (~50 MB)
  • openpyxl
  • krippendorff
  • irrCAC
After (FastAPI + HTMX)
  • fastapi (light)
  • uvicorn (light)
  • jinja2 (light)
  • python-multipart (form data)
  • openpyxl
  • krippendorff
  • irrCAC (pulls pandas transitively)

Removes ~80+ MB of direct dependencies. Install size drops dramatically.

CSS Architecture

Design Token System (replaces Quasar utility classes)
:root { /* Spacing: only these 5 values */ --ace-space-xs: 4px; --ace-space-sm: 8px; --ace-space-md: 16px; --ace-space-lg: 24px; --ace-space-xl: 32px; /* Colours */ --ace-border: #bdbdbd; --ace-bg: #ffffff; --ace-bg-muted: #f5f5f5; --ace-text: #212121; --ace-text-muted: #616161; /* Typography */ --ace-font-body: system-ui, -apple-system, sans-serif; --ace-font-mono: 'SF Mono', Menlo, Consolas, monospace; /* Timing */ --ace-transition: 0.15s; } All classes prefixed ace-. States: ace-cell--active. No BEM. No !important. No utility classes. You own it all.
